# Week 3-4 Intermediate Topics - Quick Reference

This folder contains intermediate Python topics essential for DevOps scripting.

## Days 13-20 Overview

### Day 13: Modules & Packages
- Import and use Python modules
- Create your own modules
- Organize code into packages
- **Key modules**: os, sys, subprocess, datetime

### Day 14: Working with JSON
- Parse JSON data
- Convert Python dicts to JSON
- Read/write JSON files
- Work with API responses

### Day 15: Working with YAML
- Read YAML configuration files
- Write YAML files
- Convert between YAML and Python dicts
- Use PyYAML library

### Day 16: Regular Expressions
- Pattern matching in strings
- Extract data from logs
- Validate input formats
- Search and replace text

### Day 17: Date & Time
- Work with dates and times
- Calculate time differences
- Format timestamps
- Handle timezones

### Day 18: Command Line Arguments
- Accept CLI arguments
- Use argparse module
- Build interactive CLI tools
- Parse options and flags

### Day 19-20: OOP Basics
- Classes and objects
- Methods and attributes
- Inheritance
- Create reusable components

## Learning Path

1. Complete each day's README
2. Work through practice exercises
3. Take notes in notes.md
4. Build the mini-projects

## Real-World Applications

All topics are chosen for their direct application in DevOps:
- **JSON/YAML**: Configuration management
- **Regex**: Log parsing
- **Datetime**: Scheduling and monitoring
- **CLI Args**: Building automation tools
- **OOP**: Creating reusable libraries
